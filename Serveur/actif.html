Les communications avec le serveur s’effectuent en utilisant des requêtes GET parmi  :


http://adressse-serveur/?cmd=isRaceOnProgress         
    Réponse :  True  ou  False  suivant qu’une course est en cours ou pas

http://adressse-serveur/?cmd=reinitRace         
    Effet : (re)démarre une course en (ré)initialisant tous les paramètres (liste des participants, des lieux à atteindre,…) 
    Réponse : message d’aquittement

http://adressse-serveur/?cmd=stopRace           
    Effet : arrête la course en cours sans réinitialiser ses paramètres (en particulier le nom de l’éventuel vainqueur)
    Réponse : message d’aquittement

http://adressse-serveur/?cmd=addParticipant&name=nom         
    Effet :  ajoute le participant  nom  à la liste des participants
    Réponse : message d’aquittement ou  None  si ajout impossible (pas de course en cours, participant déjà présent)

http://adressse-serveur/?cmd=removeParticipant&name=nom      
    Effet :  retire le participant   nom  de la liste des participants
    Réponse : message d’aquittement ou  None  si retrait impossible (pas de course en cours, participant absent)

http://adressse-serveur/?cmd=getParticipants          
    Réponse :  liste des participants  (noms séparés par des virgules)  ou  None  si pas de course en cours

http://adressse-serveur/?cmd=getPosition&name=nom            
    Réponse : latitude, longitude du participant ou  None  si indisponibles (pas de course en cours, participant absent)

http://adressse-serveur/?cmd=setPosition&name=nom&lat=latitude&lon=longitude
    Effet :  definit la position latitude, longitude  du participant  nom
    Réponse : message d’aquittement ou  None  si impossible (pas de course en cours, participant absent)

http://adressse-serveur/?cmd=getGoal&name=nom            
    Réponse : cible, latitude, longitude  description et coodonnées de la cible à atteindre par le participant ou  None  si indisponible

http://adressse-serveur/?cmd=setGoalReached&name=nom      
    Effet :  - si numéro de cible atteinte k = 3, définit nom comme vainqueur et arrête la course
               - sinon renvoie le numéro k  de la cible atteinte puis l'incrémente (prochaine cible à atteindre)
    Réponse : k  <= 3 numéro de la cible atteinte ou  None  si course déja terminée ou si opération impossible

http://adressse-serveur/?cmd=sendMessage&name=nom&msg=message       
    Effet :  ajoute le message  à la file d'attente du participant  nom
    Réponse : message d’aquittement ou  None  si opération impossible

http://adressse-serveur/?cmd=getMessage&name=nom      
    Effet :  retire et renvoie le message en tête de la file d'attente du participant  nom
    Réponse : message  en tête de la file ou  None  si aucune présente ou si opération impossible

http://adressse-serveur/?cmd=getWinner            
    Réponse :  nom_du_vainqueur  s’il est déterminé ou  None  sinon
